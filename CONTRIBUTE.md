## Requirements

- node >= v16 ([install](https://nodejs.org/en/download/))
- docker >= v19.03 ([install](https://docs.docker.com/get-docker/))
- docker-compose >= v1.28 ([install](https://docs.docker.com/compose/install/))

## Installation

```bash
# Create a .env file
$ cp .env.example .env
# Install dependencies
$ npm i
```

## Running the app

```bash
# Start dev
$ npm run dev
```
